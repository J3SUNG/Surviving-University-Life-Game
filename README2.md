# 대학생활 살아남기 게임

탄막 슈팅 게임 개발

## 프로젝트 소개

대학생이 되어 다가온 유혹들을 이겨내고 대학생활을 무사히 마치는 여정
유혹(게임, 잠, 술)들을 이겨내고 교수님께 좋은 학점을 받아내보자!

### 개발기간

2018.09. ~ 2018.12.

### 기술 스택

Language : Java
Tool : NetBeans

## 주요기능 목차

- 메인 화면
- 플레이 화면
- 스테이지
- 캐릭터
- 스킬
- 무기 (학과)
- 밸런스
- 랭킹
